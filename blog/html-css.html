What are the differences between relative, absolute, and fixed positioning?

The key to understanding the different types of positioning is to identify the containing block of which the target element is offset from. Here are some examples of each type below.

HTML code:
<div class="outer">Outer
    <div class="box" id="two">Two</div>
    <div class="box" id="one">One</div>
</div>
<div class="box" id="three">Three</div>

CSS code:
* {box-sizing: border-box;}
body {background-color: lightgreen;}
.box {width: 100px; height: 100px; background: red; color: white;}
#one {position: fixed; top: 50px; left: 320px; background: blue;}
#two {width: 250px; height: 250px;}
#three {width: 250px; height: 250px; background-color: purple;}
.outer {position: relative; width: 500px; height: 400px; overflow: scroll; margin: 30px; background-color: grey;}

  • Relative positioning

    Relatively positioned elements are offset a given amount from their normal position within the document, but without the offset affecting other elements.

  • Absolute positioning

    Absolutely positioned elements are positioned relative to the nearest positioned ancestor (i.e., the nearest ancestor that is not static).

  • Fixed positioning

    Fixed elements are positioned relative to the viewport, which means that they always stay in the same place even if the page is scrolled.

4. Reflections on a time I was reluctant to ask for help

I was a bit reluctant to ask about what "Super FizzBuzz" was referring to in a reflection question because my initial thought was it must be something I missed because I didn't study hard enough and the question was too minor so nobody would care.

After confirming it wasn't that case, I went ahead asking for help in Discord and one of the coaches clarified that it was outdated document which should have been updated. After that, I felt great because I found a bug and raised it which might help others as well as it helped myself.
